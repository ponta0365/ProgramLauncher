ProgramLauncher

Windows向けの常駐ランチャーです。ホットキーやグループ分けで、よく使うアプリ、スクリプト、ファイル、URLをまとめて起動できます。

最新 ZIP をダウンロード 最新 EXE をダウンロード Releases を開く

GitHub Pages を有効化すると、このページを配布案内としてそのまま公開できます。
配布物の詳細は Releases を参照してください。

主な機能

  • ホットキー起動と Alt 2回押し起動
  • 検索して即実行
  • グループ管理と一括登録
  • 自動保存、バックアップ / 復元
  • PyInstaller による EXE 配布
